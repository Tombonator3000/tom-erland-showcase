# Agent Utviklingslogg

## Prosjektoversikt
**Prosjektnavn**: Tom Erland Showcase Portfolio
**Type**: React/Vite showcase nettside
**Opprettet i**: Lovable.dev
**GitHub**: tom-erland-showcase
**Status**: Aktiv utvikling

## Teknologi Stack
- **Frontend**: React 18.3.1 + TypeScript
- **Build Tool**: Vite 5.4.19
- **UI Framework**: Tailwind CSS 3.4.17
- **UI Components**: shadcn/ui (Radix UI)
- **Routing**: React Router DOM 6.30.1
- **State Management**: TanStack Query 5.83.0
- **Styling**: Tailwind + Custom CSS animations

## Hovedfunksjoner
1. **Hero Section**
   - Parallax scrolling effects
   - Particle background
   - Typing animation for roller
   - Magnetic buttons
   - Glitch text effects

2. **Games/Portfolio Section**
   - Tilt card interactions
   - Vector War game (spillbar)
   - Shadow Realms (kommer snart)
   - Cyber Runner (kommer snart)
   - Puzzle Dimension (kommer snart)

3. **Fancy Effects**
   - Cursor glow effekt
   - Loading screen animasjon
   - Marquee for tech stack
   - Responsive design
   - Dark theme
